# Assets 目錄

請放入下列三個檔案後再執行流程：

| 檔名 | 規格 | 取得來源 |
|------|------|---------|
| `intro.mp3` | 3–5 秒，44.1kHz/128kbps | [FreePD.com](https://freepd.com) (Public Domain)、[Pixabay Music](https://pixabay.com/music/) (CC0) |
| `outro.mp3` | 3–5 秒，同規格 | 同上 |
| `cover.png` | 1400×1400 PNG，sRGB，<500KB | 自製或 [Canva](https://canva.com) 免費版 |

## Apple Podcasts 封面規格

- 尺寸：1400×1400 至 3000×3000 像素（正方形）
- 格式：JPEG 或 PNG
- 色彩空間：RGB
- 大小：建議 < 500KB
- 內容：標題清楚、無未授權商標
